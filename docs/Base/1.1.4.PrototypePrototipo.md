# 1.1.4 PrototypePrototipo

### Introdução

Para a construção do protótipo, foi utilizada a ferramenta Figma, que possibilita a criação de interfaces digitais de alta fidelidade, simulando o comportamento real do sistema. O uso do Figma permitiu a elaboração de telas interativas, facilitando a navegação entre as páginas e proporcionando uma experiência próxima ao produto final.

Foi desenvolvido um protótipo de alta fidelidade com o objetivo de representar de forma visual e interativa a solução proposta para o fórum de Inteligência Artificial. Essa etapa teve como finalidade validar rapidamente a ideia do produto antes do desenvolvimento, permitindo identificar melhorias e possíveis problemas de usabilidade.
