# 1.2.1 Léxico

O léxico é uma técnica que permite a identificação e descrição de termos e símbolos do domínio do software. Para o **Conhecendo IA**, este artefato garante que a equipe e os usuários tenham um entendimento comum sobre as funcionalidades do fórum educativo.

### Termos do Domínio

| Termo | Noção (O que é?) | Impacto (O que faz?) |
| :--- | :--- | :--- |
| **Usuário Autenticado** | Pessoa cadastrada no sistema que realizou login. | Pode criar tópicos, comentar, curtir e visualizar seu perfil. |
| **Tópico de Discussão** | Uma postagem principal criada para iniciar um debate sobre IA. | Agrupa comentários e curtidas; pode ser filtrado por categoria. |
| **Trending Topic** | O assunto mais comentado e popular do momento no fórum. | Aparece fixado no topo da Página Inicial para atrair atenção. |
| **Categoria** | Classificação temática (ex: Machine Learning, Deep Learning). | Organiza os tópicos para facilitar a navegação do usuário. |
| **Curtida (Like)** | Expressão de aprovação ou utilidade de um conteúdo. | Incrementa as estatísticas de engajamento no perfil do autor. |
| **Comentário** | Resposta ou interação enviada dentro de um tópico. | Permite a troca de conhecimento e a criação de threads de conversa. |
| **Perfil** | Espaço personalizado que consolida os dados do usuário. | Exibe foto, nome, total de comentários feitos e likes recebidos. |
| **Moderador** | Usuário com permissões especiais de administração. | Pode apagar tópicos ou comentários que violem as regras. |

---

### Metodologia e Versionamento

* **Rastreabilidade:** Este léxico baseia-se nos Requisitos Funcionais (RF01 a RF14) e nas Histórias de Usuário levantadas.
* **Senso Crítico:** A definição clara desses termos evita ambiguidades durante a modelagem BPMN e a implementação dos Padrões de Projeto.
