# 1.1.6 Introspecção

## Introdução
A técnica de introspecção consiste em os desenvolvedores imaginarem-se como usuários do sistema, identificando necessidades, expectativas, funcionalidades e possíveis melhorias com base em diferentes perfis de usuários. O sistema proposto será um fórum online voltado para discussões sobre Inteligência Artificial, permitindo que usuários compartilhem conhecimento, debatam ideias e aprendam sobre o tema.

## Metodologia
A introspecção foi realizada com base no perfil de três personas, permitindo uma visão realista sobre a criação de um fórum que busca reunir diversos perfis interessados em IA:

1. **João (Estudante de Computação):** Fã e usuário avançado de IAs, representa a parcela de usuários que busca aprofundar conhecimentos técnicos e desenvolver habilidades específicas.
2. **José do Amaral (Idoso):** Representa usuários com pouco contato com novas tecnologias que, devido ao aumento de notícias sobre o tema, procuram se informar e entender o assunto de forma simplificada.
3. **Felipe (Desenvolvedor Sênior):** Profissional experiente cuja empresa estimula a especialização em IA, representando aqueles que buscam aprendizado contínuo para aplicação no mercado de trabalho.

## Requisitos Funcionais

| ID | Requisito Funcional |
|:---:|:---|
| RF01 | O sistema deve permitir visualizar discussões recentes |
| RF02 | O sistema deve permitir filtrar discussões |
| RF03 | O sistema deve exibir trending topics |
| RF04 | O sistema deve exibir total de membros |
| RF05 | O sistema deve exibir total de tópicos ativos |
| RF06 | O sistema deve permitir navegar por categorias |
| RF07 | O sistema deve permitir criar tópicos |
| RF08 | O sistema deve permitir comentar em tópicos |
| RF09 | O sistema deve permitir responder comentários em tópicos |
| RF10 | O sistema deve permitir curtir tópicos |
| RF11 | O sistema deve permitir compartilhar tópicos |
| RF12 | O sistema deve permitir visualizar perfil |
| RF13 | O sistema deve exibir estatísticas do usuário |
| RF14 | O sistema deve permitir que o usuário se cadastre, faça login e recupere senha |
| RF15 | O sistema deve permitir pesquisar tópicos por palavras-chave ou tags |
| RF16 | O sistema deve notificar o usuário quando alguém responder ao seu comentário ou tópico |
| RF17 | O sistema deve permitir que moderadores apaguem tópicos ou comentários impróprios |
