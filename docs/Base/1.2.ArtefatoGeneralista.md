# 1.2. Artefatos Generalistas

## Introdução
O Módulo de Artefatos Generalistas compreende o conjunto de diagramas e documentos que visam fornecer uma visão macro e abstrata do sistema **ConhecendoIA**. Diferente das etapas de elicitação detalhada, estes artefatos buscam sintetizar o escopo, as intenções de design e a organização das ideias fundamentais da plataforma em um formato de fácil compreensão para todos os stakeholders.

## Objetivo
O objetivo principal deste módulo é consolidar o entendimento da equipe sobre o ecossistema do fórum de Inteligência Artificial. Através desses artefatos, é possível:
* Visualizar as conexões entre as funcionalidades planejadas;
* Estruturar a hierarquia de informações;
* Garantir que a solução proposta esteja alinhada com as necessidades das personas identificadas nas etapas de design.

Neste módulo, destaca-se o uso de técnicas visuais para mapeamento de ideias, servindo como base para a posterior modelagem de processos e arquitetura de software.
