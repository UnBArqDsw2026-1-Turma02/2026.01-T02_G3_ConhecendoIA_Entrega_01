# 1.2. Artefato Generalista

## Visão Geral do Artefato

O Rich Picture funciona como uma representação visual de baixo rigor formal, servindo para sintetizar problemas e fluxos de trabalho de maneira intuitiva. No contexto do desenvolvimento de software, ele permite que a equipe visualize como os diferentes componentes e usuários interagem entre si antes de partirmos para modelagens mais complexas. O diagrama foca em elementos centrais: quem são os usuários, quais ações eles executam, onde os dados repousam e quais os limites de atuação do nosso software.
